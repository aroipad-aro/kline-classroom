# K 線型態互動教室

捏 K 棒的互動教學網頁：用手指拖拉 K 棒的開高低收，即時看型態名稱、多空意義與位置情境（低檔／盤整／高檔）解讀。

三個分頁：
- **單根 K 棒**：十字、鎚子、流星、光頭光腳……16 種單根型態
- **兩根組合**：吞噬、母子、烏雲罩頂、貫穿線、鑷頂鑷底……11 種經典組合
- **三根組合**：晨星、夜星、紅三兵、三隻烏鴉、內困／外側三日……8 種經典組合

- 單一 `index.html`，無任何相依套件，手機優先設計
- 線上版（GitHub Pages）：https://aroipad-aro.github.io/kline-classroom/

## 手機安裝方式

1. 用 Safari 開上面的連結
2. 點「分享」→「加入主畫面」
3. 之後就能像 App 一樣從主畫面開啟
